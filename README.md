# Servlet-JSP-JSTL-and-MySQL-Projects

This project demonstrated HttpServlet Scopes
    - Application Scope (Context Scope),
    Session Scope,
    Request Scope
    
and how to forward and redirect between servlets. Using JSP for only view perspective.
